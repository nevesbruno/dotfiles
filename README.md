# Dot files

My personal configurations including MacOSX Tweak.

## Features

#### install.sh

Extract the files prepending the dot character to make it hidden.

#### macosx.sh

Tweak to make MacOSX more enjoyable.

## Usage

``` shell
git clone git@github.com:wbotelhos/dotfiles.git
./dotfiles/install.sh
./dotfiles/macosx.sh
```
